# Personal Music Showcase
A music streaming webcase that showcases the music I have produced and released to the public. Users can scroll through my songs and listen live on the webpage. Each song has a dedicated link to its corresponding spotify location.

https://carterhanford.github.io/music-showcase/

## Table of Contents
- img 
  - a folder for all music artwork images on the webpage
- music
  - wav files for all music on the webpage
- index.html
  - html code for webpage setup
- script.js
  - javascript code for functionality
- style.cs
  - css code for style of webpage
